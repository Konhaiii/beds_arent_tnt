## Beds Aren’t TNT

Ever tried to sleep in the Nether or the End and accidentally turned your bed into a bomb? This mod fixes that.

![Beds Aren't TNT](https://cdn.modrinth.com/data/cached_images/58f86b13e9184159e55c1df3a2aec43378586394.jpeg)

Beds Aren’t TNT prevents beds from exploding in the Nether and the End. Instead of blowing up, the game simply displays a message explaining that you cannot sleep in these dimensions. No more accidental explosions, destroyed bases, or unexpected deaths.
